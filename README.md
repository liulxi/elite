# elite
名校无忧手机端：
	1、熟练运用rem技术布局
	2、参考UI给的psd 默认尺寸，本项目是750px, 平板显示的时候最大宽度显示750px
	3、手机端实现的特效：
		1> 侧边栏特效
		2> 2个弹出效果
			a、点击左下角指纹出现一排QQ、微信、信息、空间、置顶图标
			b、点击右下角手机图片、居中显示黄色背景的对话框并做关闭隐藏，5s后又自动显示
		3> 图片切换
		4> 留学头条 向上循环滚动
		5> 热门留学院校做地区切换
		6> 热门留学院校 “点击加载更多” 
				出现一排内容后 出现文字”点击收起更多“ 并点击返回
		7> 优选留学方案 下拉显示（注意改变上下的提示箭头）
